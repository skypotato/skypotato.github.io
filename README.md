## Jekyll 사용 환경 구성

### 전제조건
- [Ruby 설치](https://www.ruby-lang.org/en/downloads/)
- [MinGW 설치](http://www.mingw.org/)
- [Make 설치](http://gnuwin32.sourceforge.net/packages/make.htm)

### 사용 환경 확인
```shell script
ruby -v #Ruby
gem -v #RubyGems
gcc -v #GCC <- window 환경에서는 MinGW 설치
g++ -v #GCC <- window 환경에서는 MinGW 설치
make -v #Make
```
