# TDD

## Red Green Cycle
- Red : 항상 실패하는 테스트를 먼저 작성한다.
- Green : 테스트가 통과하는 프로덕션 코드를 작성한다.
- Refactor : 테스트가 통과하면 프로덕션 코드를 리팩토링한다.
